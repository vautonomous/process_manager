# process_manager
This package manage system processes with ROS messages

Table 1. Commands

| Commands |     Process      |
|:--------:|:----------------:|
|    0     |   Shutdown PC    | 
|    1     |    Reboot PC     |
|    2     |  Start Autoware  |
|    3     | Restart Autoware |
|    4     |  Kill Autoware   |
